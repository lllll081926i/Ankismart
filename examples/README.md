## 使用方法 / How to Use

### 方法一：通过界面加载 / Method 1: Load via UI
1. 在导入页面点击"加载示例"按钮
2. 选择想要使用的示例文档
3. 系统会自动加载文件并推荐合适的生成策略

### 方法二：直接拖拽 / Method 2: Drag and Drop
1. 打开文件管理器，导航到 `examples` 目录
2. 将示例文件拖拽到 Ankismart 的文件选择区域
3. 手动配置生成策略

## 技巧提示 / Tips

- **文档长度**: 建议单个文档 2000-5000 字，便于生成高质量卡片
- **内容结构**: 使用清晰的标题层级，帮助 AI 理解内容结构
- **重点标记**: 使用加粗、引用等格式突出重点内容
- **代码示例**: 为代码块添加语言标识，提高可读性

- **Document Length**: Recommended 2000-5000 words per document for quality cards
- **Content Structure**: Use clear heading hierarchy to help AI understand structure
- **Highlight Key Points**: Use bold, quotes to emphasize important content
- **Code Examples**: Add language identifiers to code blocks for better readability
