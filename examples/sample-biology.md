# 生物学知识示例 / Biology Knowledge Examples

## 细胞生物学 / Cell Biology

### 细胞结构 / Cell Structure

**真核细胞的主要组成部分：**

1. **细胞核（Nucleus）**
   - 包含遗传物质 DNA
   - 控制细胞活动
   - 核膜具有核孔，调控物质进出

2. **线粒体（Mitochondria）**
   - 细胞的"能量工厂"
   - 进行有氧呼吸
   - 产生 ATP（三磷酸腺苷）

3. **内质网（Endoplasmic Reticulum）**
   - 粗面内质网：附着核糖体，合成蛋白质
   - 光面内质网：合成脂质，解毒

4. **高尔基体（Golgi Apparatus）**
   - 修饰、包装和运输蛋白质
   - 形成分泌囊泡

5. **核糖体（Ribosomes）**
   - 蛋白质合成的场所
   - 由 rRNA 和蛋白质组成

### 细胞膜 / Cell Membrane

**流动镶嵌模型**：细胞膜由磷脂双分子层构成，蛋白质镶嵌其中，具有流动性。

**主要功能：**
- 选择性通透
- 细胞识别
- 信号转导
- 物质运输

## 遗传学 / Genetics

### DNA 结构 / DNA Structure

**双螺旋结构特点：**
- 两条反向平行的多核苷酸链
- 碱基配对原则：A-T（腺嘌呤-胸腺嘧啶），G-C（鸟嘌呤-胞嘧啶）
- 氢键连接碱基对：A-T 两个氢键，G-C 三个氢键
- 右手螺旋，直径约 2nm

### 中心法则 / Central Dogma

```
DNA → RNA → 蛋白质
 ↓      ↓
复制   转录   翻译
```

**关键过程：**

1. **DNA 复制（Replication）**
   - 半保留复制
   - DNA 聚合酶催化
   - 5' → 3' 方向合成

2. **转录（Transcription）**
   - DNA → mRNA
   - RNA 聚合酶催化
   - 在细胞核中进行

3. **翻译（Translation）**
   - mRNA → 蛋白质
   - 核糖体上进行
   - tRNA 携带氨基酸

### 孟德尔遗传定律 / Mendelian Inheritance

**分离定律**：等位基因在配子形成时分离，每个配子只含一个等位基因。

**自由组合定律**：不同基因的等位基因自由组合，互不干扰。

**遗传比例示例：**
- 单基因杂交 F2 代：3:1（显性:隐性）
- 双基因杂交 F2 代：9:3:3:1

## 生态学 / Ecology

### 生态系统组成 / Ecosystem Components

**生物成分：**
1. **生产者**：绿色植物、光合细菌（自养生物）
2. **消费者**：草食动物、肉食动物、杂食动物
3. **分解者**：细菌、真菌（腐生生物）

**非生物成分：**
- 阳光、温度、水分
- 空气、土壤
- 无机盐类

### 能量流动 / Energy Flow

**特点：**
- 单向流动，不可循环
- 逐级递减
- 能量传递效率约 10-20%

**营养级：**
```
生产者 → 初级消费者 → 次级消费者 → 三级消费者
(植物)   (草食动物)    (小型肉食)    (大型肉食)
```

### 物质循环 / Material Cycles

**碳循环：**
- 大气中的 CO₂
- 光合作用固定碳
- 呼吸作用释放 CO₂
- 化石燃料燃烧

**氮循环：**
- 固氮作用（固氮菌、根瘤菌）
- 硝化作用（硝化细菌）
- 反硝化作用（反硝化细菌）
- 氨化作用（分解者）

## 进化论 / Evolution

### 达尔文自然选择学说 / Darwin's Natural Selection

**核心观点：**
1. **过度繁殖**：生物繁殖能力超过环境承载力
2. **生存斗争**：个体间竞争有限资源
3. **遗传变异**：个体间存在可遗传的差异
4. **适者生存**：适应环境的个体存活并繁殖

### 现代综合进化论 / Modern Synthesis

**进化的基本单位**：种群

**进化的机制：**
- 突变：提供原始材料
- 基因重组：产生新的基因型
- 自然选择：决定进化方向
- 隔离：形成新物种的必要条件

### 物种形成 / Speciation

**地理隔离 → 生殖隔离 → 新物种形成**

**生殖隔离类型：**
- 生态隔离
- 季节隔离
- 行为隔离
- 机械隔离
- 配子隔离
- 杂种不育

## 人体生理学 / Human Physiology

### 神经系统 / Nervous System

**神经元结构：**
- 细胞体
- 树突（接收信号）
- 轴突（传导信号）
- 突触（传递信号）

**神经冲动传导：**
- 静息电位：约 -70mV
- 动作电位：去极化 → 复极化
- 传导方向：单向传导
- 传导速度：有髓鞘神经纤维更快（跳跃式传导）

### 内分泌系统 / Endocrine System

**主要内分泌腺：**

| 腺体 | 激素 | 主要功能 |
|------|------|---------|
| 垂体 | 生长激素 | 促进生长发育 |
| 甲状腺 | 甲状腺激素 | 调节代谢 |
| 胰岛 | 胰岛素 | 降低血糖 |
| 肾上腺 | 肾上腺素 | 应激反应 |
| 性腺 | 性激素 | 生殖发育 |

### 免疫系统 / Immune System

**三道防线：**

1. **第一道防线**：皮肤、黏膜（非特异性免疫）
2. **第二道防线**：体液中的杀菌物质、吞噬细胞（非特异性免疫）
3. **第三道防线**：特异性免疫（体液免疫、细胞免疫）

**特异性免疫：**
- **体液免疫**：B 细胞 → 浆细胞 → 抗体
- **细胞免疫**：T 细胞 → 效应 T 细胞 → 直接杀伤靶细胞

## 植物生理学 / Plant Physiology

### 光合作用 / Photosynthesis

**总反应式：**

$$6CO_2 + 6H_2O \xrightarrow{光能} C_6H_{12}O_6 + 6O_2$$

**两个阶段：**

1. **光反应**（类囊体膜）
   - 水的光解
   - ATP 和 NADPH 的生成

2. **暗反应**（叶绿体基质）
   - CO₂ 固定（卡尔文循环）
   - 有机物合成

### 呼吸作用 / Cellular Respiration

**有氧呼吸：**

$$C_6H_{12}O_6 + 6O_2 \rightarrow 6CO_2 + 6H_2O + 能量(ATP)$$

**三个阶段：**
1. 糖酵解（细胞质）
2. 柠檬酸循环（线粒体基质）
3. 电子传递链（线粒体内膜）

**能量产出**：1 分子葡萄糖完全氧化产生约 38 个 ATP

### 植物激素 / Plant Hormones

**五大类植物激素：**

1. **生长素（Auxin）**：促进细胞伸长，顶端优势
2. **赤霉素（Gibberellin）**：促进茎伸长，打破休眠
3. **细胞分裂素（Cytokinin）**：促进细胞分裂
4. **脱落酸（ABA）**：抑制生长，促进休眠
5. **乙烯（Ethylene）**：促进果实成熟
