# 基础输入控件示例

包含各种基础输入控件的演示代码。

## 🎛️ 组件列表

- **button/** - 各种按钮组件示例
  - `demo.py` - PushButton, ToolButton, PrimaryPushButton 等按钮类型演示

- **check_box/** - 复选框组件示例
  - `demo.py` - CheckBox 复选框演示

- **combo_box/** - 组合框组件示例
  - `demo.py` - ComboBox 下拉选择框演示

- **model_combo_box/** - 模型组合框组件示例
  - `demo.py` - ModelComboBox 基于模型的组合框演示

- **radio_button/** - 单选按钮组件示例
  - `demo.py` - RadioButton 单选按钮演示

- **slider/** - 滑块组件示例
  - `demo.py` - Slider 滑块控件演示

- **switch_button/** - 开关按钮组件示例
  - `demo.py` - SwitchButton 开关按钮演示

运行 `python demo.py` 查看每个组件的演示效果。
