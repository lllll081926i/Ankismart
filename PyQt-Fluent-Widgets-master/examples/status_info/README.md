# 状态信息显示组件示例

包含各种状态信息提示组件的演示代码。

## 📊 组件列表

- **info_badge/** - 信息徽章示例
  - `demo.py` - InfoBadge 信息徽章演示

- **info_bar/** - 信息栏示例
  - `demo.py` - InfoBar 信息栏演示

- **progress_bar/** - 进度条示例
  - `demo.py` - ProgressBar 进度条演示

- **progress_ring/** - 进度环示例
  - `demo.py` - ProgressRing 进度环演示

- **state_tool_tip/** - 状态工具提示示例
  - `demo.py` - StateToolTip 状态工具提示演示

- **tool_tip/** - 工具提示示例
  - `demo.py` - ToolTip 工具提示演示

运行 `python demo.py` 查看状态信息组件效果。
