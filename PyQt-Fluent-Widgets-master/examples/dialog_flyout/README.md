# 对话框和浮层组件示例

包含各种对话框和浮层提示组件的演示代码。

## 🗨️ 组件列表

- **color_dialog/** - 颜色选择对话框示例
  - `demo.py` - ColorDialog 颜色选择器演示

- **custom_message_box/** - 自定义消息框示例
  - `demo.py` - 自定义消息框演示

- **dialog/** - 通用对话框示例
  - `demo.py` - Dialog 对话框演示

- **flyout/** - 浮层提示示例
  - `demo.py` - Flyout 浮层提示演示

- **folder_list_dialog/** - 文件夹列表对话框示例
  - `demo.py` - FolderListDialog 文件夹列表对话框演示

- **message_dialog/** - 消息对话框示例
  - `demo.py` - MessageDialog 消息对话框演示

- **teaching_tip/** - 教学提示示例
  - `demo.py` - TeachingTip 教学提示演示

运行 `python demo.py` 查看每个组件的演示效果。
