# 文本相关组件示例

包含文本显示和输入组件的演示代码。

## 📝 组件列表

- **font_icon/** - 字体图标示例
  - `demo.py` - FontIcon 字体图标演示

- **image_label/** - 图片标签示例
  - `demo.py` - ImageLabel 图片标签演示

- **label/** - 标签示例
  - `demo.py` - Label 标签演示

- **line_edit/** - 单行文本框示例
  - `demo.py` - LineEdit 单行文本框演示

- **spin_box/** - 数字调节框示例
  - `demo.py` - SpinBox 数字调节框演示

- **text_browser/** - 文本浏览器示例
  - `demo.py` - TextBrowser 文本浏览器演示

运行 `python demo.py` 查看文本组件效果。
