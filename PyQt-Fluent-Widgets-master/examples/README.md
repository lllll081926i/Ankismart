# PyQt-Fluent-Widgets 示例代码

这是一个包含所有 PyQt-Fluent-Widgets 组件示例的目录，按功能分类组织。

## 📁 目录结构

- **basic_input/** - 基础输入控件示例
- **date_time/** - 日期时间选择器示例
- **dialog_flyout/** - 对话框和浮层组件示例
- **gallery/** - 综合示例应用，展示所有组件的使用
- **layout/** - 布局相关示例
- **material/** - 材质效果组件示例
- **media/** - 多媒体组件示例
- **menu/** - 菜单和工具栏示例
- **navigation/** - 导航组件示例
- **scroll/** - 滚动区域相关示例
- **status_info/** - 状态信息显示组件示例
- **text/** - 文本相关组件示例
- **view/** - 视图组件示例
- **window/** - 窗口组件示例

每个子目录都包含对应组件的演示代码，可以通过运行 `python demo.py` 来查看效果。
