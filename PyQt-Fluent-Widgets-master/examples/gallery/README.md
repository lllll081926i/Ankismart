# 综合示例应用

这是一个完整的综合示例应用，展示 PyQt-Fluent-Widgets 所有组件的使用方法和效果。

## 🎨 特点

- **完整示例** - 包含所有组件的演示界面
- **主题切换** - 支持明暗主题切换
- **多语言** - 支持中文和英文界面
- **现代化界面** - 采用 Fluent Design 设计语言

## 📋 目录结构

- **app/** - 应用主程序目录
  - `common/` - 公共配置和资源文件
  - `components/` - 自定义组件
  - `view/` - 各个功能页面
  - `main.py` - 应用入口文件

- **demo.py** - 快速演示入口
- **gallery.pro** - Qt 项目文件

## 🚀 运行方法

```bash
cd examples/gallery
python demo.py
```

这将启动一个完整的示例应用，展示所有可用的组件和功能。
