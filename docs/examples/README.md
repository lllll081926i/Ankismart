# Ankismart 示例文档说明

本目录包含用于测试和学习 Ankismart 功能的示例文档。

## 文件列表

### sample.md

**描述**：包含丰富数学公式的 Markdown 示例文档

**内容**：
- 基础数学运算（加减乘除、指数、对数）
- 代数（方程、不等式、方程组）
- 微积分（极限、导数、积分）
- 线性代数（向量、矩阵）
- 概率统计（概率、分布）
- 几何（平面几何、三角函数）
- 数论（整除、同余）
- 集合论（集合运算）
- 求和与乘积
- 希腊字母和特殊符号

**适用场景**：
- 学习如何在 Ankismart 中使用 LaTeX 数学公式
- 测试数学公式渲染效果
- 生成数学相关的 Anki 卡片

**推荐策略**：
- **基础问答**：适合概念性问题
- **关键术语**：适合公式和定义
- **单选题**：适合数学计算题

**使用方法**：
1. 在 Ankismart 中点击"导入"
2. 选择 `sample.md` 文件
3. 选择生成策略（推荐"基础问答"或"关键术语"）
4. 点击"开始生成"
5. 在预览页面查看生成的卡片
6. 检查数学公式是否正确渲染
7. 导出到 Anki 或保存为 .apkg 文件

### 数学公式示例

```markdown
# 微积分基础

## 导数

导数表示函数的变化率：

$$
f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}
$$

常见导数公式：
- 幂函数：$(x^n)' = nx^{n-1}$
- 指数函数：$(e^x)' = e^x$
- 对数函数：$(\ln x)' = \frac{1}{x}$
```

## 最佳实践

### 文档结构

1. **使用清晰的标题层级**：
   - `#` 一级标题（文档标题）
   - `##` 二级标题（章节）
   - `###` 三级标题（小节）

2. **合理组织内容**：
   - 每个章节包含相关的概念
   - 使用列表突出要点
   - 添加示例和说明

3. **适当的内容密度**：
   - 不要过于简短（信息不足）
   - 不要过于冗长（难以提取关键点）
   - 每个章节 200-500 字为宜

### 数学公式

1. **行内公式**：用于文本中的简短公式
   ```markdown
   勾股定理：$a^2 + b^2 = c^2$
   ```

2. **独立公式**：用于重要的、需要突出显示的公式
   ```markdown
   $$
   \int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
   $$
   ```

3. **公式编号**（可选）：
   ```markdown
   $$
   E = mc^2 \tag{1}
   $$
   ```

### 代码块

    print("Hello, World!")
